# Onboarding Checklist — WhatsApp AI Agent

## Pre-briefing (before day 1)
- [ ] Proposta aceita + pagamento 50% recebido
- [ ] Acesso ao WhatsApp Business do cliente (API ou QR scan)
- [ ] Contato principal definido (quem responde dúvidas no setup)

## Briefing (day 1, ~30min call)
- [ ] Mapear serviços/produtos principais
- [ ] Tabela de preços/FAQ (cliente fornece ou extraímos do site/cardápio)
- [ ] Horários de funcionamento
- [ ] Regras de escalonamento (quando transferir para humano)
- [ ] Tom de voz (formal/informal, emojis, tratamento)
- [ ] Integrações necessárias (agenda, CRM, planilha)

## Configuração (days 2-3)
- [ ] Prompt do agente configurado
- [ ] Base de conhecimento carregada (FAQ, preços, serviços)
- [ ] Fluxos de conversa testados internamente
- [ ] Agendamento integrado (se aplicável)
- [ ] Escalonamento configurado

## Teste (days 4-5)
- [ ] Cliente testa como "cliente fantasma"
- [ ] Ajustes de tom/conteúdo baseados no feedback
- [ ] Teste de escalonamento
- [ ] Teste fora do horário

## Go-live (days 6-7)
- [ ] Agente ativado em produção
- [ ] Monitoramento ativo primeiras 48h
- [ ] Pagamento 50% restante
- [ ] Relatório da primeira semana agendado

## Pós go-live
- [ ] Check-in dia 3 (ajustes finos)
- [ ] Relatório semana 1 (métricas + sugestões)
- [ ] Relatório mês 1 (ROI + próximos passos)
